# Garion Projetos — WordPress Maintenance Toolkit

Plugin WordPress para otimização, limpeza e manutenção de sites.

## Funcionalidades planejadas

- Limpeza de revisões
- Remoção de transients expirados
- Otimização de banco de dados
- Desativação de recursos desnecessários
- Controle da Heartbeat API
- Limpeza de cache
- Painel de diagnóstico
- Verificação de versão do PHP
- Validação de HTTPS
- Checagem de extensões (Imagick, Intl)

## Requisitos

- WordPress 6.x+
- PHP 8.0+
- MySQL 5.7+ / MariaDB
